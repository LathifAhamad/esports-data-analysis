# 🎮 Esports Data Analysis & Visualization Project

## 📌 Overview

This project focuses on analyzing esports data to uncover trends in game earnings, player engagement, and tournament activity using Python.

The workflow includes:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Statistical Testing
* Machine Learning Model

---

## 📊 Dataset Features

* Game
* ReleaseDate
* Genre
* TotalEarnings
* OfflineEarnings
* PercentOffline
* TotalPlayers
* TotalTournaments

---

## 📈 Key Visualizations

* Line Chart → Earnings trend over years
* Bar Chart → Top games by earnings
* Pie Chart → Genre distribution
* Scatter Plot → Players/Tournaments vs Earnings
* Box Plot → Outlier detection
* Heatmap → Correlation analysis
* Area Plot → Growth trend
* Stacked Bar Chart → Genre-wise earnings

---

## 🔍 Key Insights

* Few games dominate total esports earnings
* Higher player base leads to increased revenue
* Tournaments significantly influence earnings
* Data is skewed with noticeable outliers
* Strong correlation between players and earnings

---

## 📊 Statistical Analysis

* Pearson Correlation Test
* T-Test (High vs Low earning games)

---

## 🤖 Machine Learning

* Model: Linear Regression
* Features: TotalPlayers, TotalTournaments
* Target: TotalEarnings
* Evaluation: R² Score
